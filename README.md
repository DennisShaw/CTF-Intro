# CTF-Intro（持续更新中……）
## CTF是什么

CTF（Capture The Flag）起源于 1996 年 **DEFCON** 全球黑客大会 ，在网络安全领域中指的是网络安全技术人员之间进行技术竞技的一种比赛形式
